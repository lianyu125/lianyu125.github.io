```python
range(1,6) #代表从1到6(不包含6)
```
[1, 2, 3, 4, 5]
```python
range(1,6,2) #代表从1到6，间隔2(不包含6)
```
[1, 3, 5]
```python
range(6) #代表从0到6(不包含6)
```
## 数组的操作
```python
array = [1, 3, 5, 7, 9, 2, 4, 6, 8, 0]
array[0:] #列出0以后的
```
[1, 3, 5, 7, 9, 2, 4, 6, 8, 0]
```python
array[1:] #列出1以后的
```
[3, 5, 7, 9, 2, 4, 6, 8, 0]
```python
array[:-1] #列出-1之前的
```
[1, 3, 5, 7, 9, 2, 4, 6, 8]
```python
array[3:-3] #列出3到-3之间的
```
[7, 9, 2, 4]
```python
array[::2] #[1, 5, 9, 4, 8]
```
```python
array[::3]#[1, 7, 4, 0]
```
```python
array[::-1]#数组逆序,[0, 8, 6, 4, 2, 9, 7, 5, 3, 1]
```
