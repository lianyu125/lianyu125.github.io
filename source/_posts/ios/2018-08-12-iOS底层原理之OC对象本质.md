## Objective-C的本质
我们平时编写的Object-C代码，底层都是C/C++代码，所以Objective-C的面向对象都是基于C/C++的数据结构实现的
【注】Objective-C的对象、类主要是基于C/C++的结构体实现的
### NSObject的底层实现

```objc
@interface  NSObjec {
  Class isa;
}
```
C语言实现
```c/c++
struct NSObject_IMPL {
	Class isa;
};
```
