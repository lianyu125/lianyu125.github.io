PHP Socket的使用

技术 PHP 2014/01/25

Socket是什么

socket的英文原义是“孔”或“插座”。作为BSD UNIX的进程通信机制，取后一种意思。通常也称作”套接字”，用于描述IP地址和端口，是一个通信链的句柄。在Internet上的主机一般运行了多个服务软件，同时提供几种服务。每种服务都打开一个Socket，并绑定到一个端口上，不同的端口对应于不同的服务。

以上内容来自百度百科

简单说来，socket可以帮助不同的服务在不同的端口进行通信。

PHP中的实现

服务端

服务端主要进行了以下步骤：

  • 创建Socket的监听，等待连接
  • 当链接到来时，开启一个子连接处理IO
  • 接收来自客户端的传输数据
  • 将结果写回给客户端

客户端

客户端主要有以下步骤：

  • 连接到服务端Socket
  • 向服务端写数据
  • 接收来自服务端的数据
